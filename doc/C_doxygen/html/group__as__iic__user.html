Execute a read transaction for a single byte. Writes the slave (read) address and receives the byte the slave sends. Automatically converts the slave write address to the read address, if the write address is given.

Parameters
base_addrAddress of the corresponding as_iic hardware module (see also as_hardware.h)
slave_addrThe address of the slave on the bus to talk to.
read_dataPointer to a byte variable to copy the byte the slave send to.
Returns
Byte containing a status code, 0 if OK.

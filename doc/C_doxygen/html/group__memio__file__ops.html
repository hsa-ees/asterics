Performs data transfers from hardware to software by utilizing the as_memwriter module. Calls may be blocking or nonblocking by using the O_NONBLOCK flag upon "open". In case the following function blocks the process, it is woken up by the interrupt handler (currently timer-based). Otherwise, "as_memio_hw_update" is called on a regular basis by the interrupt handler.

static int driver_as_mmap ( struct file *  filp,
struct vm_area_struct *  vma 
)
static

Maps a physically concurrent memory area (acquired by control device) to virtual address space of the user (driver_as_mmap). Transfers data between user application and hardware without copying the data (as_mmap_ioctl). It uses one of the memory modules of the existing memio devices for transferring data.

◆ vears_overlay_set_color()

void vears_overlay_set_color ( uint32_t  vears_iobase,
int  col_idx,
uint32_t  color 
)

Set entry of the overlay color map.

Parameters
vears_iobaseI/O address of the VEARS module
col_idxColor index (1..3)
colorColor in 0xrrggbb format

If no color is set, the default colors are:

  • 1: red
  • 2: green
  • 3: blue

Note: This call has no effect if the VEARS module is not enabled.

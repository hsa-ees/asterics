Detailed Description

Class that represents a 2D Window Pipeline of the ASTERICS Framework.

Inheriting the AsModuleGroup class, this class describes a 2D Window Pipeline subsystem. Containes additional attributes managing the window interfaces (as_automatics_2d_window_interface::AsWindowInterface) and image row buffers (as_automatics_2d_pipeline_row::AsPipelineRow) of the subsystem. Contains custom connect methods for the modules within the subsystem and methods for creating, managing and optimizing the size of all image row buffers.

Detailed Description

Automatic's own graph class - keeps track of nodes added to the graph.

Sadly no nice interface for checking for added nodes exists in graphviz (not that I'm aware of, anyways). The default behaviour of the graphviz API is to automatically add nodes that don't exist when adding edges - we don't want this here!

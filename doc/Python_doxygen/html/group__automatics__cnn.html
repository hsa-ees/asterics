◆ new_nn_layer()

AsNNLayer asterics.new_nn_layer ( int  image_width,
int   image_height = 480,
str   name = "" 
)

Provide a new AsNNLayer object.

This allows implementation of a neural network layer using a 2D Window Pipeline. You need to provide the width and optionally height of the input image. Provide a unique name, if you want to implement multiple layers. After creating a new layer object using this method, use 'layer.parametrize_and_build()' to configure the layer.

Parameters
image_widthThe number of horizontal pixels per image row
image_heightThe number of vertical pixels per image column
nameThe name of the layer
Returns
A new AsNNLayer object.

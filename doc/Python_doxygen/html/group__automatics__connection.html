◆ resolve_generic()

bool as_automatics_connection_helper.resolve_generic ( Port  port)

Make sure Generics within port's data width exist in its parent module.

This method makes sure that the Generics in port's data width exist in the VDHL entity of port's entity. Port may also be a GlueSignal Process: 1. Extract Generic strings from port's data width

  1. Match those strings to Generic objects in Port and the parent module
  2. Check if the Generic value is set; Check for and match linked Generic
  3. If necessary, substitute the Generic(s) with a match in group module
  4. If possible, use the defined value of the Generic
  5. Update port's data width and try to evaluate it.
    Parameters
    portThe data width attribute of this port will be resolved.
    Returns
    True if the resolve function ran, False if nothing was done.

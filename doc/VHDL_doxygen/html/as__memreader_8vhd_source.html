129  -- Allows for "dynamic" configuration of slave registers
130  -- Possible values and what they mean:
131  -- "00": Register "off". Register will not be available and won't be implemented -> No hardware resource utilization.
132  -- "01": From HW view: Status register. Data transport from hardware to software. HW can only write, SW can only read.
133  -- "10": From HW view: Control register. Data transport from software to hardware. HW can only read, SW can only write.
134  -- "11": Combined Read/Write register. Data transport in both directions.
135  -- When both sides attempt to write simultaniously, only the HW gets to write.

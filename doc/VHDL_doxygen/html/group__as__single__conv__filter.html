Detailed Description

This module packages 'as_2d_conv_filter_internal' as an AsStream pixel streaming module. One of the following convolution kernels can be selected using generics: Laplace edge filter ('laplace'), Sobel X or Y edge filter ('sobel_x', 'sobel_y'), Gauss blurring filter ('gauss').

ASTERICS VHDL Helper Modules

This Doxygen group contains all ASTERICS VHDL packages and helpers modules not specific to any ASTERICS hardware module. More...

Modules

 AXI_Master: ASTERICS AXI Master Adapter
 A customized AXI Master implementation for the ASTERICS Framework. This AXI Master is based on the code generated by Xilinx Vivado. This files content was generated by the Xilinx IP Wizard and modified to build an encapsulated AXI-Master interface.
 
 AXI_Slave: ASTERICS AXI Slave Adapter
 A customized AXI Slave implementation for the ASTERICS Framework. This AXI Slave is based on the code generated by Xilinx Vivado. This files content was generated by the Xilinx IP Wizard and modified to build an encapsulated AXI-Master interface.
 
 fifo_fwft: First Word Fall Through FIFO
 First Word Fall Through FIFO: If data is written into the empty fifo, it is presented on the output ports one clock cycle later. The read_in port confirms the data at the output ports. The FIFO will present the next data package, if available.
 
 as_fifo: First In First Out Buffer
 Simple First In First Out buffer. Depends on as_misc/.../ram/DUAL_BRAM_READ_FIRST.vhd.
 
 as_generic_filter: Convolution Filter
 This entity implements a generic convolution filter with arbitrary weights.
 
 as_line_buffer: Image Row Buffer
 This module implements the internal image row buffer used in 2D Window Pipelines.
 
 as_pipeline_manager: 2D Window Pipeline Manager
 Implements a manager component for pipeline flushing, stride and stall behaviour management.
 
 as_pipeline_row: Image Row Buffer with Filter Window
 This entity implements the image row buffer used in 2D Window Pipelines. Provides a buffer output and a row of the image filter window. Implements the buffer memory either as registers or using BRAM.
 
 as_shift_line: Register-based FIFO
 Implementation of a FIFO as a shift register.
 
 window_buff_nxm: Standalone Window Buffer
 Standalone 2D Window Pipeline - style window buffer. Implements a N by M window buffer. Meant mainly for testing purposes, such as testbenches.
 
 as_cnn_helpers: ASTERICS Helper Package for CNN Hardware Modules
 This package contains constants, functions and data types used frequently throughout VHDL code of modules implementing functionality related to CNNs in ASTERICS.
 
 as_filter_mask: ASTERICS Helpers for Canny Pipeline
 This helper package contains data types and functions used by 2D Window Pipeline modules used in the Canny edge detector system.
 
 as_sim_ram_pkg: ASTERICS Helpers for Canny Pipeline
 This helper package contains constants, data types and functions used by the simulated memory reader and writer modules for debugging ASTERICS systems.
 
 helpers: Standard ASTERICS Helper Package
 This package contains constants, functions and data types used frequently throughout VHDL code in ASTERICS.
 
 BRAM_READ_FIRST: Read-first BRAM
 Implements BRAM which is performs a read operation before a write operation.
 
 BRAM_WRITE_FIRST: Write-first BRAM
 Implements BRAM which performs a write operation before a read operation.
 
 DUAL_BRAM_READ_FIRST: Read-first Dual-Port BRAM
 Implements a dual-ported BRAM which performs read operations before write operations.
 
 DUAL_BRAM_READ_WRITE: Dual-Port BRAM
 Implements dual-ported BRAM which can perform read and write operations simultaneously.
 
 ram: Dual-Port BRAM
 Implements dual-ported BRAM.
 
 as_generic_regslice: ASTERICS Slave Register
 This module implements a single configurable 32 bit register slice for use with the register manager module 'as_regmgr.vhd'.
 
 as_regmgr: ASTERICS Slave Register Manager
 This module implements the registers for any module within an ASTERICS system. It instantiates and manages a configurable number of 'as_generic_regslice' modules, each a single 32 bit register with potentially different read/write access.
 
 as_sim_image_reader: as_memreader Simulation Module
 This module simulates the behaviour of an as_memreader module for hardware simulation.
 
 as_sim_file_writer: as_memwriter Simulation Module
 This module simulates the behaviour of an as_memwriter module for hardware simulation.
